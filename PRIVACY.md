# Privacy Note

VS Translator Overlay is an external helper app for Vintage Story.

- It does not modify Vintage Story files.
- It does not inject into the game process.
- It reads the local Vintage Story `client-chat.log` file.
- It can capture a screen area selected by the user for OCR.
- It uses online translation through the configured translator library, so translated text may be sent to third-party translation services.
- Runtime config, logs, and OCR debug files are stored in `%APPDATA%\VS Translator Overlay`.

Do not OCR or translate sensitive text if you do not want it sent to an online translator.

---

# Примечание о приватности

VS Translator Overlay - это внешнее вспомогательное приложение для Vintage Story.

- Оно не изменяет файлы Vintage Story.
- Оно не внедряется в процесс игры.
- Оно читает локальный файл Vintage Story `client-chat.log`.
- Оно может захватывать выбранную пользователем область экрана для OCR.
- Оно использует онлайн-перевод через настроенную библиотеку переводчика, поэтому переводимый текст может отправляться сторонним сервисам перевода.
- Настройки, логи и отладочные файлы OCR хранятся в `%APPDATA%\VS Translator Overlay`.

Не распознавайте через OCR и не переводите чувствительный текст, если не хотите отправлять его онлайн-переводчику.
