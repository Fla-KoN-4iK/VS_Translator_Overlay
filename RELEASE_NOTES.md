# VS Translator Overlay v0.1.0

First public prototype release.

## Highlights

- Incoming Vintage Story chat translation from `client-chat.log`.
- Outgoing message translation with preview, edit, copy, and paste.
- Windows OCR screen/sign translation.
- Settings dialog for chat log path, languages, OCR language, overlay opacity, font size, and message count.
- Portable one-folder Windows build.

## Known Limitations

- Online translation quality depends on the translator service.
- OCR quality depends on the installed Windows OCR language pack and image clarity.
- `Alt+Q` is quick mode; `Alt+S` is recommended for more accurate OCR because it lets you select the exact screen region.
